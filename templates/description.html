Mini Project 2: Image Classification with a Custom CNN

This challenge is part of the Mini Project 2 assignment. Participants are required to implement, train, and evaluate a Convolutional Neural Network (CNN) model for an image classification task. The dataset contains images grouped into class folders, and the goal is to correctly classify each image into its corresponding category.
